Function Types (#3)

Implement mapIndexedNotNull() function by hand. Like mapNotNull() it applies the given transformation to each element and filters out nulls. Like mapIndexed() it bases the transformation on both the element and its index.

Note the generic types R? (in (Int, T) -> R?) and List<R>. Lambda argument may return null as a result, so lambda's return type is nullable R?. mapIndexNotNull() returns a list of non-nullable elements, so the function return type is List<R>. To express that R is a non-nullable type, we specify the constraint on a generic type parameter R : Any. Type constraints will be covered in [Generics] later.
